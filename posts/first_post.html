This is my first post! A bit of a test really.

The idea is:

  1. Create markdown file
  2. Convert to html
  3. Include it in the viewer

There are some benefits

  • Easier to write, no messing with html tags
  • Markdown :DDDDDDDDDDDDDDDDDDDDDD
  • Easier to add custom stuff once it’s in html form

How to convert?

I’m thinking of using pandoc to convert between these. Here be a sample script 

#!/usr/bin/env sh

# Get the filename without the extension
infn="$1"

basename="$(basename "$infn")"

outfn="$(echo "$basename" | sed 's/.md$//' )"

pandoc --from markdown "$infn" --to html -o "${outfn}.html"
